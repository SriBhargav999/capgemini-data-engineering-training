**Objective**
  The objective of this assignment is to strengthen SQL fundamentals by practicing aggregation, grouping, and different types of joins.
  It focuses on understanding how to analyze data and combine multiple tables to extract meaningful insights.
_____________________________________________________________________________________________________________________________________________
**Problem Statement (Summary)**

  This assignment covers two major areas:
  
  GROUP BY & Aggregation:
  - Calculate total, average, minimum, and maximum values
  - Count records based on different conditions
  - Apply filtering using HAVING clause
  - Perform aggregations with conditions
  - Solve real-world analytical problems
  
  JOINS:
  - Perform INNER, LEFT, RIGHT, and FULL OUTER joins
  - Combine data from multiple tables (employees, departments, projects)
  - Handle NULL values in joins
  - Identify missing or unmatched records
  - Solve real-world join-based scenarios

  6. Solved real-world problems involving both aggregation and joins

**Key Concepts Covered**
  - GROUP BY – grouping data for aggregation
  - HAVING – filtering aggregated results
  - Aggregate functions – SUM, COUNT, AVG, MIN, MAX
  - INNER JOIN – matching records from both tables
  - LEFT JOIN – all records from left table with matches
  - RIGHT JOIN – all records from right table with matches
  - FULL OUTER JOIN – all records from both tables
  - Handling NULL values in joins
